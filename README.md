Welcome to the Code-legion-Alpha repository. This project serves as a structured workspace for daily coding practice, technical experiments, and building foundational software components.

 🚀 Project Overview
Code-legion-Alpha is designed to track progress through consistent, daily development. Each "Day" folder contains specific implementations, logic exercises, or project milestones.

 📁 Repository Structure
The repository is organized into daily progress folders


 🛠️ Language used [Java]
Tools: Git, GitHub Languages VS Code,

📝 How to Use
1.  Clone the repository:
    ```bash
    git clone [https://github.com/kfabulous/Code-legion-Alpha.git](https://github.com/kfabulous/Code-legion-Alpha.git)
    ```
2.  Navigate to a specific day:
    ```bash
    cd Code-legion-Alpha/Day1
    ```
3.  Run the code for each: Follow the specific instructions inside each folder's sub-directory.
